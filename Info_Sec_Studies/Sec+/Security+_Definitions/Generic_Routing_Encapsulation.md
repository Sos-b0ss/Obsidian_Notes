Generic Routing Encapsulation is a method of encapsulation of [[IP]] packet in a [[GRE]] header which hides the original [[IP]] packet. Also a new header named delivery header is added above [[GRE]] header which contains new source and destination address. [[GRE]] header act as new [[IP_Header]] with Delivery header containing new source and destination address.
